css 글씨 다루기

  • font-size

    글씨의 크기를 지정한다.

  • font-weight

    글씨의 가중도(굵기)를 지정한다.

  • text-align

    글씨의 정렬방법을 지정한다.

    Lorem ipsum dolor sit, amet consectetur adipisicing elit. Saepe vel hic nam optio incidunt repudiandae dolore natus temporibus assumenda quae?

  • line-height

    줄의 높이를 지정한다.

    Lorem ipsum dolor sit amet, consectetur adipisicing elit. Explicabo inventore beatae voluptatem ullam vero, tempore neque iure incidunt natus totam saepe delectus. Nisi totam dolorem porro? Expedita autem maiores mollitia?

  • word-spacing, letter-spacing

    단어, 글자, 간의 간격을 지정한다.

  • text-transform

    단어를 변형시킨다.

  • text-decoration

    단어를 밑줄, 취소선 등으로 꾸민다.
