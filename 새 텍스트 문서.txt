함수작성
객체에 대해서 이해

할당, 참조, 호출, 식별자, 데이터 타입을 등 용어

할당(asignment): 변수에 값을 저장하는거

선언(declaration) : 식별자를 만든다.

참조(reference): 식별자 (변수나 함수 혹은 객체의 프로퍼티)에 저장된 값을 잃어오는것

호출(call): 작성된 함수를 살행하는것

반환 (return): 함수의 실행 결과로 특정값을 준다.

매개변수(Parameter): 함수에 전달된 값에 이름을 짓는 것(함수 내부에서 사용할 변수와 비슷)

인자(agument) :매개변수를 통해서 실제로 값

분기처리 : 조건에 따라서 할생될 코드를 작성하는것

프로퍼티(property) :객체 할당되어 있는 속성 => ex) user.name

메서드(method) : 객체에 할당되어 있는 속성이 함수일 때 매서드라고 한다.

객체 : { } (객체 리터럴)을 통해 작성한 데이터를 혹은 new 연산자와 함깨 생성자를 호출하면 변환되는 값
	=> .을 통해 프로퍼티 참조가 가능하면 객체
	=> 객체가 아닌 (원시값): 문자(string), 숫자(number), 불리언 (boolean: turnm false), undefined, null
	=> JS에서 문자열. strsub() 처럼 매서드 호툴이 가능하지만 객체가 아니가

콜백 함수: 함수에 인자로 전달되는 함수
	=> 상황에 따라 실행될 코드가 다를 때 사용한다.
	=> ex) 클릭시 실행될 코드를 함수형태로 전달 

스코프(scope) :식별자의 유효 범위
